# Calculator (Skill)

## Quick start

```json
{
  "expression": "2 + 3 * 4"
}
```

## Behavior

- Accepts digits and basic arithmetic operators: `+ - * / ( ) %`.
- Rejects expressions containing other characters.

## Output

```json
{
  "expression": "string",
  "result": "string"
}
```
